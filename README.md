## How to download the project:

```bash
# Clone the repository
$ git clone https://github.com/rodrigoaraujo7/finances-app.git

# Enter directory
$ cd finances-app
```

## Usage

1. Run `yarn`.<br />
2. Run `expo start` and press `a` to open your android emulator.<br />
